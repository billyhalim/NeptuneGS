NEPTUNEGS

A Simple Grid System for Beginners

WHY USE NEPTUNEGS ?

EASY TO USE

NeptuneGS provides you built-in CSS classes that you can just use them in your HTML's class attribute.

RESPONSIVE

NeptuneGS helps you to make websites that fit devices viewport (screen size). The built-in CSS classes are available in different screen size.

24-GRID COLUMN

NeptuneGS has 24-grid column to make your website more presentable and clean.
